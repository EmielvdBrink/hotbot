usage: cowsay [option] message
    --alt <arg>    Alt text for HTML output mode, '%s' will be replaced
                   with the message
 -b                Initiates Borg mode
 -d                Causes the cow to appear dead
 -e <arg>          Select the appearance of the cow's eyes
 -f <arg>          Specifies a cowfile to use
 -g                Invokes greedy mode
 -h                Display this help message
    --html         HTML output mode
 -l                To list all cowfiles
    --lang <arg>   Change language, e.g. "lang de" for German
 -n                The given message will not be word-wrapped
 -p                Causes a state of paranoia to come over the cow
 -s                Makes the cow appear thoroughly stoned
 -t                Yields a tired cow
 -T <arg>          Select the appearance of the cow's tongue
 -W <arg>          Specifies roughly (where the message should be wrapped.
                   The default is equivalent to -W 40
 -w                Initiates wired mode
 -y                Brings on the cow's youthful appearance
